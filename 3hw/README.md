Решалось следующее задание:
1. Выберите любой открытый датасет и скачайте открытый датасет, соответствующий вашим интересам или области обучения.
2. Создайте новую базу данных в системе управления базами данных (например, SQLite, PostgreSQL).
3. Создайте таблицу (или несколько таблиц) в базе данных с различными типами данных (INTEGER, TEXT, DATE), которые требуются для вашего датасета. Импортируйте данные из датасета в созданные таблицы.
4. Напишите несколько SQL-запросов для извлечения данных из таблиц базы данных. Используйте условия фильтрации (например, WHERE) для получения нужных данных.
5. Напишите SQL-запросы, использующие агрегатные функции (SUM, AVG, COUNT) для выполнения расчетов по данным таблицы.
6. Визуализируйте данные. Используйте библиотеки Python, такие как Matplotlib или Seaborn, для визуализации данных, извлеченных из базы данных. Постройте графики или диаграммы, которые помогут проанализировать и понять данные.

Для работы был взят набор данных об успеваемости студентов на экзаменах https://www.kaggle.com/datasets/grandmaster07/student-exam-performance-dataset-analysis

- **`Hours_Studied`** — количество часов, затраченных на учебу (числовой).
- **`Attendance`** — посещаемость занятий в процентах (числовой).
- **`Parental_Involvement`** — уровень вовлеченности родителей (категориальный): `Low` — низкий, `Medium` — средний, `High` — высокий.
- **`Access_to_Resources`** — доступ к образовательным ресурсам (категориальный): `Low` — низкий, `Medium` — средний, `High` — высокий.
- **`Extracurricular_Activities`** — участие во внеклассных занятиях (категориальный): `Yes` — да, `No` — нет.
- **`Sleep_Hours`** — среднее количество часов сна в сутки (числовой).
- **`Previous_Scores`** — баллы за предыдущие экзамены (числовой).
- **`Motivation_Level`** — уровень мотивации студента (категориальный): `Low` — низкий, `Medium` — средний, `High` — высокий.
- **`Internet_Access`** — наличие доступа к интернету (категориальный): `Yes` — да, `No` — нет.
- **`Tutoring_Sessions`** — количество дополнительных занятий с репетитором в месяц (числовой).
- **`Family_Income`** — уровень дохода семьи (категориальный): `Low` — низкий, `Medium` — средний, `High` — высокий.
- **`Teacher_Quality`** — качество преподавания (категориальный): `Low` — низкое, `Medium` — среднее, `High` — высокое.
- **`School_Type`** — тип учебного заведения (категориальный): `Public` — государственное, `Private` — частное.
- **`Peer_Influence`** — влияние сверстников на учебу (категориальный): `Positive` — положительное, `Neutral` — нейтральное, `Negative` — отрицательное.
- **`Physical_Activity`** — количество часов физической активности в неделю (числовой).
- **`Learning_Disabilities`** — наличие проблем с обучением (категориальный): `Yes` — да, `No` — нет.
- **`Parental_Education_Level`** — уровень образования родителей (категориальный): `High School` — средняя школа, `College` — колледж, `Postgraduate` — высшее образование.
- **`Distance_from_Home`** — расстояние от дома до учебного заведения (категориальный): `Near` — близко, `Moderate` — умеренно, `Far` — далеко.
- **`Gender`** — пол студента (категориальный): `Male` — мужской, `Female` — женский.
- **`Exam_Score`** — **целевая переменная**: итоговый балл за экзамен, который необходимо предсказать (числовой).
