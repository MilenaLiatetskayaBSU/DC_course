# Итоговый проект 
## Обзор выбранных данных
Для выполнения работы был выбран датасет(временной ряд), который содержит исторические данные о торгах акциями Microsoft (https://www.kaggle.com/datasets/vijayvvenkitesh/microsoft-stock-time-series-analysis).
Данные предоставлены за период 04.01.2015 — 04.01.2021. 

## Структура данных

| Колонка | Описание | 
|---------|----------|
| `Date` | Дата торгов | 
| `Open` | Цена открытия |
| `High` | Максимальная цена за день |
| `Low` | Минимальная цена за день | 
| `Close` | Цена закрытия |
| `Volume` | Объем торгов (шт.) |
